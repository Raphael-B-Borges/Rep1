Brings you closer to
what you love

With less than a month left until the release of Windows 11, Microsoft started a promotion campaign for the upcoming operating system this Thursday (9). Called "Windows 11: Brings you closer to what you love", the video published on the company's YouTube channel extols the platform's unprecedented features in an unusual way — in the best possible sense.

Top 3 Jobs

In the announcement, Microsoft also clarified that upcoming "technical" builds of Windows 11 will be numbered higher than those found in the beta channel builds. This change came after the company authorized the distribution of versions still in active development to the developer channel, reflecting a "more updated" experience than expected for the release of the new operating system.

Developer-oriented version 22454 introduces discrete changes to the context menu when interacting with the "Recycle Bin" shortcut, displaying options that match the new Windows 11 aesthetic. Similarly, the "File Explorer" context menu will have an option to pin shared files over the network in the "Quick Access" section — no extra clicks required, as in the previous version.

According to Microsoft, the update will be divided into phases, in a "learning of Windows 10". Thus, the first to receive the Windows 11 update will be the new eligible devices. The company's expectation is that "all eligible devices will receive the free upgrade to Windows 11 by mid-2022". On qualifying Windows 10 PCs, Windows Update should alert the user when the new version is available.

Making the ISO available

Installing without update
Download

The official Windows 11 ISO can be downloaded directly from the Microsoft website. For those who are going to perform a clean installation of Windows 11 on the same computer that the ISO was downloaded from, it is not necessary to create a bootable pendrive, it is possible to do this only with the image file of the new OS.

Safe source

The recommen- dation is to download the ISO directly from the Microsoft website. The Windows 11 image is available for download with the company's official tool and does not need an activation key to be downloaded or installed.

Creating

If you have already downloaded a Windows 11 ISO on yours and want to create a USB stick using the file, it is possible to start Windows 10. With this, it is not necessary to download or have internet access to perform the procedure on the computer.

Updates and fixes

Some improvements
Tabs

The new update for Windows 11, called “Moment 1” by Microsoft, was released four weeks after the debut of Windows 11 22H2. It brings a long-awaited feature that has been missing for years: tabs in File Explorer

Contextual menu

The famous “Moment 1” presented other novelties, such as the presence of a more extensive contextual menu (the famous right mouse button) or better management of applications in the taskbar, among others.It is worth mentioning that this patch is optional and will not be installed automatically. To get it, it's simple, just go to the "Update and Security" section in the "Settings" application and click "download and install".

Android on CPU

Microsoft plans to add Android 13 functions to Windows 11 as part of future plans, this is an attempt to move more Android elements to the CPU OS. This is an iterative update built on top of Android 12, with a "much more elaborate Material You design, multi-tasking tools, and cross-compatibility with Chromebook and Google tablets", among others.

Windows 11 and internet

We don't always have an internet via Wi-Fi nearby, but we usually have a cell phone in hand. It is with this in mind that Microsoft has released a new feature in the latest Windows 11 Insider Preview Build 25231. In this way, the Phone Link app allows instant connection between the mobile internet of the Samsung smartphone with the user's PC.

Now, with the new version of the test build of Windows 11, companies have added the functionality to turn on the mobile internet from the smartphone and connect it to the PC without having to touch the device. However, only Samsung devices with One UI 4.1.1 or later can be used for this function.

According to Microsoft, when the cell phone is close to the computer, its network will appear among the Wi-Fi options. The user simply clicks on the access point to initiate the connection. That is, there will no longer be the need to enter passwords or change device settings more than once. Finally, you can connect from the “What's New” screen or from the “Link to Mobile” app settings.
